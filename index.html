Introduction

CSS transitions allow to control the speed on which CSS properties can change.

In fact by default such variations take place instantly, while using CSS transitions these variations take pace on specific time intervals that follows an acceleration curve, that is completely customizible.

CSS transitions allow you to define:

  • which properties to animate (transition-property);
  • when the animation will start (transition-delay);
  • the duration of the transition (transition-duration);
  • how the transition will take place (transition-timing-function).
Transition-property

It allows to define the name or the names of the CSS properties on which you want to apply the transitions.

It’s important to note that only certain properties can be animated using CSS transitions.

On this link you can find an exhaustive list of animatable CSS properties.

.box { transition-property: width; }
Transition-duration

It allows to specify the duration (in seconds or in milliseconds) of the transitions.

In the case of acting on more properties, it’s possible to set a single duration for all the properties, or different durations for each property involved.

Note that it’s essential to define the duration of a transition, because the default value is zero.

.box { transition-duration: 2s; }
Transition-timing-function

It allows to define the speed curve that the transition wil follow.

The transition-timing-function property can have the following values:

  • ease: it’s the default value, slow start, then fast, then end slowly
  • linear: same speed from start to end;
  • ease-in: slow start
  • ease-out: slow-end
  • easy-in-out: slow start and slow end;
  • cubic-bezier(n,n,n,n): it allows to customize the speed curve according to your own requirements.
.box1 {transition-timing-function: linear;} .box2 {transition-timing-function: ease;} .box3 {transition-timing-function: ease-in;} .box4 {transition-timing-function: ease-out;} .box5 {transition-timing-function: ease-in-out;}
Transition-delay

As the name suggests, it allows defining a delay (in seconds) for the transition event.

.box { transition-delay: 1s; }
Transition

The transition property is a shorthand property used to represent up to four transition-related longhand properties:

.box { transition: background-color 800ms ease-in-out 2s; }
Vendor Prefixes

For compatibility in all supporting browsers, vendor prefixes are required.

Note that the standard syntax is declared at last:

.box { -webkit-transition: background-color 800ms ease-in-out 2s; -moz-transition: background-color 800ms ease-in-out 2s; -o-transition: background-color 800ms ease-in-out 2s; transition: background-color 800ms ease-in-out 2s; }
